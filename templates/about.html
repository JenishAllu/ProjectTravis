About HERMES

HERMES (Hearing-Enabled Response and Multilingual Echo System) is an advanced, accessible, multilingual live chat platform designed to empower visually impaired users and support seamless communication for everyone. Developed by Team Travis G242, HERMES leverages state-of-the-art speech recognition and synthesis, AI-powered translation, and robust accessibility features to ensure that no one is left behind in digital conversations.

What is HERMES?

HERMES is your intelligent, voice-driven assistant for real-time, inclusive communication. Users can interact in English, Hindi, or Telugu—by typing or speaking—and receive instant responses, with all messages translated as needed. HERMES is built to be accessible for the visually impaired, offering high-contrast UI, full keyboard navigation, and screen reader compatibility. Speech-to-text and text-to-speech work in all supported languages, making chat accessible to everyone.

How to Use

  1. Register as a customer or agent, selecting your preferred language (English, Hindi, Telugu).
  2. Login with your credentials.
  3. Customers: Start chatting with an agent or AI. Type or use the mic to send voice messages in your language.
  4. Agents: View all online customers, join chats, and reply using text or voice. All controls are accessible via keyboard and screen reader.
  5. Messages are automatically translated and transcribed, with chat history securely stored for your convenience.

Key Features

  • Real-time chat between customers and agents using WebSockets (Socket.IO)
  • Multilingual support: automatic translation and speech recognition for English, Hindi, and Telugu
  • AI-powered intent classification and smart automated replies
  • Speech-to-text and text-to-speech for all supported languages
  • Screen reader and keyboard navigation for visually impaired users
  • Secure authentication and role-based access (customer/agent)
  • Persistent, searchable chat history with MongoDB
  • Modern, high-contrast, and responsive UI
  • Privacy-focused: no data shared with third parties

Keyboard Shortcuts

  • Alt+Space: Hold to record (mic on while holding)
  • Alt+S: Send message
  • Alt+A: Open About page/modal
  • Alt+L: Focus customer list and read first customer
  • Up/Down Arrow (in customer list): Move selection and read customer name
  • Enter (in customer list): Select highlighted customer
  • Esc: Exit input or customer list navigation
  • Alt+C: Read chat from bottom
  • Left/Right Arrow: Navigate chat messages

Accessibility: HERMES is designed for everyone. All controls are keyboard-accessible, and both customers and agents can use screen readers. After voice transcription, the result is spoken aloud for visually impaired users. The interface is high-contrast, easy to navigate, and supports TTS and STT in multiple languages. HERMES is committed to digital inclusion and independence for all users.

About the Name

HERMES – Hearing-Enabled Response and Multilingual Echo System
Hermes, the messenger of the gods in Greek mythology, symbolizes voice, communication, speed, guidance, and helping souls find their way. This makes HERMES the perfect name for a system that empowers visually impaired users to navigate digital conversations and tasks through speech and multilingual support.

Why Choose HERMES?

  • Empowers visually impaired users with seamless voice and text chat
  • Supports English, Hindi, and Telugu for both speech and text
  • Ensures privacy and security for all users
  • Open-source and community-driven
  • Continuously improved for accessibility and inclusion
